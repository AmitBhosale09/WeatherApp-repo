# WeatherApp

Simple Java web application (Servlet) that shows weather information.

Project layout
- `src/main/java` — Java source (`MyPackage/MyServlet.java`)
- `src/main/webapp` — web resources (`index.html`, `index.jsp`, `style.css`, `myScript.js`)

How to run
- Build a WAR and deploy to Apache Tomcat 9+ or run from your IDE (Eclipse/IntelliJ).

Quick steps (manual):
1. Compile and package into a WAR (or use your IDE export).
2. Copy the WAR to Tomcat's `webapps/` folder and start Tomcat.
